There are basically Two types of Lists in HTML

  1. Ordered List: - To create Ordered list The tag 'ol' is Used

    • It can be nested by itself and by Unordered list as well

      • Unordered list is being nested in itself (Item 1)

      • Unordered list is being nested in itself (Item 2)

    • This is the example of nesting Unordered list in the Ordered list

  2. Unordered List: - To create Unordered list The tag 'ul' is Used

    1. Ordered list is being nested in itself (Item 1)
    2. Ordered list is being nested in itself (Item 2)
